Privacy Policy

Last Updated: December 1, 2025

Introduction

Deal Bandit ("we", "our", or "the Extension") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our Chrome extension.

Key Point: We do not collect, store, or transmit your personal data to any external servers. All data processing happens locally in your browser.

Information We Collect

Deal Bandit collects and stores the following information locally in your browser:

  • API Keys: Your OpenAI and SerpApi keys (if provided) are stored locally in Chrome's storage.
  • Usage Statistics: Number of product analyses performed (for freemium model).
  • Account Information: Email address (if you create an account for unlimited analyses).
  • Analysis History: Previously analyzed products and results (stored locally).

How We Use Your Information

Your information is used solely for:

  • Analyzing marketplace listings using AI (OpenAI GPT-4)
  • Searching for market prices (SerpApi)
  • Tracking usage for freemium limitations (5 free analyses)
  • Storing your preferences and settings

Data Storage

All data is stored locally using Chrome's chrome.storage.local API. Your data remains on your device and is NOT sent to Deal Bandit servers.

Important: API keys stored in Chrome's local storage are not encrypted by default. Anyone with physical or remote access to your computer could potentially access these keys. We recommend using API keys with spending limits and monitoring usage.

Third-Party Services

Deal Bandit integrates with the following third-party services:

OpenAI API

  • Purpose: AI-powered product analysis and price evaluation
  • Data Shared: Product titles, descriptions, prices, and images from marketplace listings
  • Privacy Policy: https://openai.com/privacy/

SerpApi

  • Purpose: Real-time market price searches
  • Data Shared: Product names and search queries
  • Privacy Policy: https://serpapi.com/privacy

Facebook Marketplace

  • Purpose: Scraping product listing information
  • Data Collected: Publicly visible listing data (title, price, description, images)
  • Note: We convert publicly accessible images to base64 format for AI analysis

Data Security

We implement the following security measures:

  • All API keys remain on your local device
  • No data is transmitted to Deal Bandit servers
  • HTTPS is used for all external API communications
  • Error boundaries prevent sensitive data from appearing in error logs

Your Rights

You have the right to:

  • Delete all stored data by uninstalling the extension
  • Clear your API keys at any time through the settings
  • View what data is stored using Chrome's Developer Tools (chrome://extensions)
  • Opt out of the freemium model by not creating an account

Children's Privacy

Deal Bandit is not intended for users under the age of 13. We do not knowingly collect personal information from children under 13.

Changes to This Policy

We may update this Privacy Policy from time to time. We will notify users of significant changes by updating the "Last Updated" date at the top of this policy.
